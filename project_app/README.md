# قدها — Project App

واجهة تطبيق **قدها**: منصة عربية لقياس الملاءة المالية قبل الدخول في التزامات جديدة مثل التمويل أو التقسيط. يركز هذا المجلد على تطبيق الواجهة الأمامية وتجربة المستخدم، مع محاكاة رحلة المصرفية المفتوحة وتحليل التدفق النقدي وإصدار **شهادة قدها**.

## الفكرة

تساعد **قدها** المستخدم على فهم قدرته المالية من خلال تحليل الدخل، المصاريف، الالتزامات، واستقرار الرصيد. بعد ذلك تعرض المنصة مؤشرًا واضحًا للملاءة المالية، وتوصيات لتحسين الوضع المالي، وشهادة مختصرة قابلة للمشاركة دون كشف الحساب الكامل.

## الواجهات الرئيسية

- **لوحة التحكم:** ملخص المؤشرات المالية ومؤشر قدها.
- **ربط الحسابات:** محاكاة ربط حساب بنكي بعد موافقة المستخدم.
- **تحليل التدفق المالي:** عرض الدخل، المصاريف، الالتزامات، والفائض النقدي.
- **محاكي الالتزام:** اختبار أثر قسط أو تمويل جديد على القدرة المالية.
- **شهادة قدها:** عرض نتيجة الملاءة وإصدار شهادة مختصرة.
- **شات قدها:** مساعد ذكي لشرح النتائج والإجابة عن الأسئلة المالية.
- **خطة التحسين المالي:** توصيات عملية لتحسين الملاءة قبل التقديم.

## التقنيات المستخدمة

- **React + Vite** لبناء تطبيق ويب سريع.
- **TypeScript** لتنظيم الكود وتقليل الأخطاء.
- **Tailwind CSS** لتصميم واجهات عربية RTL متجاوبة.
- **React Router** لإدارة التنقل بين الصفحات.
- **Recharts** لعرض الرسوم والمؤشرات المالية.
- **Google GenAI** لتشغيل قدرات المساعد الذكي عند توفر مفتاح API.

## تشغيل المشروع محليًا

المتطلبات:

- Node.js
- npm

خطوات التشغيل:

```bash
cd project_app
npm install
cp .env.example .env.local
npm run dev
```

بعد التشغيل افتح:

```text
http://localhost:3000
```

## متغيرات البيئة

أنشئ ملف `.env.local` داخل مجلد `project_app` وأضف القيم التالية:

```env
GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
APP_URL="http://localhost:3000"
```

## أوامر مفيدة

```bash
npm run dev      # تشغيل بيئة التطوير
npm run build    # بناء نسخة الإنتاج
npm run preview  # معاينة نسخة الإنتاج
npm run lint     # فحص TypeScript
npm run clean    # حذف ملفات البناء المؤقتة
```
